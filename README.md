## Installation
The code was developped using Python 3.7.9. Necessary packages are:
- glob
- time
- random
- functools
- pandas
- numpy
- seaborn
- matplotlib
- scikit-learn

The environment can be recreated using the **requirements.txt** or **environment.yml** file in this repository.
